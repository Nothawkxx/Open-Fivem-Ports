# 🔓 FiveM VPS Port Opener

A simple and effective batch script to configure your VPS for FiveM servers. This script automatically opens the required ports for both CFX and txAdmin using Windows Firewall.

## 🚀 Features

* Opens **30120 (TCP/UDP)** for CFX/FiveM
* Opens **40120 (TCP/UDP)** for txAdmin
* Fully automated setup
* No manual firewall configuration needed

## ⚙️ Usage

1. Upload the `.bat` file to your VPS
2. Right-click and **Run as Administrator**
3. Done — your ports are ready!

## ⚠️ Requirements

* Windows VPS
* Administrator privileges

Quick, clean, and perfect for fast server deployment.
